Without using javascript, plugin has tremendous features like choice of using table in 20 + variant colors, 100% responsive saves you from unnecessary headache, very intuitive administration panel and most importantly implementation using simple shortcodes.

== Installation ==

= Installation needs only few simple steps: =

 = 1) Unzip hostrider-pricing-tables.zip archive = 
 = 2) Log into your WordPress administration panel = 
 = 3) Open the "Plugins" submenu = 
 = 4) Click "Add new" button = 
 = 5) Click "Upload" link = 
 = 6) Select hostrider-pricing-tables.zip archive from unpacked folder (hostrider-pricing-tables folder) = 
 = 7) Click "Install now" = 
 = 8) After installation process click "Activate" link =
